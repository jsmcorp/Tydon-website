Frequently Asked Questions

Is this app affiliated with the government? +
No. Ration Shop Manager is a private product by Tydon Technologies Pvt. Ltd. It is not affiliated with any government body. We have submitted a formal API request to the Maharashtra Food Department and NIC for official data integration.
How does the app get my shop's data? +
The app uses your FPS shop code to retrieve data from publicly accessible government PDS portals — the same data visible on IMPDS and ePoS websites. We are working toward official API integration with government approval.
Is my data safe? +
Yes. All data is encrypted in transit and stored securely on Microsoft Azure servers in India. Your data is never shared with third parties.
Does the app work in Marathi and Hindi? +
Hindi and Marathi support is on our roadmap. Current version is in English with simple, easy-to-understand interface designed for FPS dealers.
What if the government portal changes? +
Our technical team monitors portal changes and updates the app immediately. We are actively pursuing official API access for long-term stability.
